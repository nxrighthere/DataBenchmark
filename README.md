# DataBenchmark

Performance on AMD FX-4300 with `-DNDEBUG` using GCC 8.1.0:

| DOD     | OOD     | OOD (Fragmented) | OOP (Inheritance) |
|---------|---------|------------------|-------------------|
| 107.224 | 102.277 | 2027.3           | 1850.76           |

Optimized compilation `-O3` breaks the benchmark.
